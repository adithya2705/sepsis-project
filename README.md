# 🏥 ICU Sepsis Early Warning System

## 📌 Overview
This project predicts sepsis risk using Machine Learning.

## 🚀 Features
- Low / Moderate / High risk prediction
- Uses clinical patient data
- Real-time Streamlit app

## 🧠 Models Used
- XGBoost
- LSTM
- Ensemble Learning

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- TensorFlow
- XGBoost

## ▶️ Run Project
```bash
pip install -r requirements.txt
streamlit run app.py
